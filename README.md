# Facebook-Food-Trends-Engagement-Dashboard
End-to-end analysis of Facebook food engagement data using Python (EDA) and Power BI dashboards to uncover customer insights.
# FoodTrends: Facebook Engagement Analysis

## Project Overview
This project analyzes Facebook food-related posts to understand customer engagement patterns using Python and Power BI.

## Features
- Data Preprocessing
- Exploratory Data Analysis (EDA)
- Data Visualization
- Interactive Power BI Dashboard

## Tools Used
- Python (Pandas, NumPy)
- Matplotlib & Seaborn
- Power BI

## Dataset
Contains Facebook food posts with engagement metrics such as likes, comments, shares, and reactions.
